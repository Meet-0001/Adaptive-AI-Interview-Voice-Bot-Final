# Adaptive-AI-Interview-Voice-Bot-Final
🎙️ AI-powered voice interview bot that adapts questions in real-time based on candidate responses using local LLMs via Ollama &amp; Flask.

# 🎯 InterviewMentor AI

An AI-powered interview coach that runs **100% locally** on your machine. Practice real interview questions, get instant feedback, speak your answers out loud, and track your progress — all without sending your data to the cloud.

---

## ✨ Features

- 🤖 **AI Question Generation** — Generates behavioral, technical, and situational questions tailored to your role, experience level, and difficulty
- 📝 **AI Answer Evaluation** — Scores your answers (1–10) with detailed feedback on relevance, clarity, and depth
- 🎙️ **Speech-to-Text** — Speak your answers using OpenAI Whisper (runs locally)
- 🔊 **Text-to-Speech** — Questions are read aloud using Microsoft Edge TTS
- 📊 **Interview History** — Saves up to 10 past sessions with scores and full data
- 🔒 **100% Local & Private** — No API keys, no cloud, everything runs on your machine

---

## 🚀 Getting Started

### Prerequisites

- [Docker Desktop](https://www.docker.com/products/docker-desktop/) installed and running
- Windows 10/11

---

### Installation

**1. Clone or download this project**

**2. Open CMD inside the project folder**
- Open File Explorer and navigate to the project folder
- Click the address bar, type `cmd`, press Enter

**3. Build and start the containers**
```bat
docker compose up -d --build
```

**4. Pull the AI model (first time only, ~2GB)**
```bat
docker exec -it ollama ollama pull llama3.2
```

**5. Open your browser**
```
http://127.0.0.1:5002
```

---

### One-Click Launch (after first setup)

1. Make sure **Docker Desktop** is running
2. Double-click **`start.bat`**
3. Browser opens automatically ✅

To stop everything:
- Double-click **`stop.bat`**

---

### Auto-start Docker Desktop with Windows

1. Open Docker Desktop
2. Go to **Settings → General**
3. Enable **"Start Docker Desktop when you sign in"**
4. Click **Apply & Restart**

After this, just double-click `start.bat` after booting your PC — no manual steps needed.

## 🎮 How to Use

1. **Select your role** — e.g. Software Engineer, Data Scientist, Product Manager
2. **Set difficulty** — Very Easy / Easy / Medium / Difficult
3. **Choose experience level** — Junior / Mid / Senior
4. **Pick question types** — Behavioral, Technical, Situational
5. **Start the interview** — Questions are generated by the local AI
6. **Answer by typing or speaking** — Use the mic button for voice input
7. **Get instant feedback** — Score, strengths, improvements, and coaching tips
8. **Review history** — See all past sessions and scores

## 🔌 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/` | Serves the frontend |
| GET | `/api/health` | Health check for all services |
| GET | `/api/ollama-models` | Lists available Ollama models |
| POST | `/api/generate-questions` | Generates interview questions |
| POST | `/api/evaluate-answer` | Evaluates and scores an answer |
| POST | `/api/stt` | Speech-to-text transcription |
| POST | `/api/tts` | Text-to-speech audio |
| POST | `/api/save-session` | Saves interview session |
| GET | `/api/history` | Retrieves interview history |
| DELETE | `/api/history/<id>` | Deletes a session |

## ⚙️ Configuration

You can change these values at the top of `app.py`:

| Variable | Default | Description |
|----------|---------|-------------|
| `DEFAULT_MODEL` | `llama3.2` | Ollama model to use |
| `TTS_VOICE` | `en-US-GuyNeural` | Edge TTS voice |
| `MAX_INTERVIEWS` | `10` | Max saved sessions |
